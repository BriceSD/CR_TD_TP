Client RichejQuery >> Plugin

Un plugin jQuery est tout simplement une nouvelle méthode (fonction membre) que nous ajoutons à l'objet .prototype de la fonction jQuery.

De cette façon, tous les objets fabriqués par la fonction jQuery, que nous appelons tableaux jQuery dans ce cours, hériteront de cette méthode supplémentaire.

Au lieu d'utiliser jQuery.prototype ou $.prototype, jQuery utilise un alias qui s'écrit $.fn.
En effet, le prototype est avant tout un objet qui va contenir les fonctions-méthodes qui pourront être héritées par les objets créés par jQuery.
L'abréviation fn pour fonctions semble donc plus adaptée.

Le but d'un plugin est de faire quelque chose avec une collection d'éléments du DOM.

Chaque méthode fournie avec le noyau de jQuery, telle que .fadeOut () ou .addClass (), peut être considérée comme un plugin.

Des informations sur la manière d'utiliser ou de créer un plugin peuvent être trouvées à l'adresse : http://learn.jquery.com/plugins/

Pour lister les plugins existant, rendez-vous à l'adresse :
http://plugins.jquery.com/
